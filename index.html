¿Para qué sirven HTML y CSS?

HTML y CSS son dos lenguajes de marcado utilizados para crear páginas web. HTML (HyperText Markup Language) se utiliza para estructurar y presentar el contenido de una página web, mientras que CSS (Cascading Style Sheets) se utiliza para aplicar estilos y diseños a la página.

HTML proporciona la estructura básica de una página web, como los encabezados, párrafos, listas, tablas, imágenes y formularios. También se utiliza para definir la relación entre diferentes partes de la página, como los enlaces entre páginas y los marcadores para la navegación.

CSS se utiliza para dar estilo a una página web, incluyendo la fuente, el color, el tamaño y la posición del texto, las imágenes y otros elementos de la página. CSS también se utiliza para controlar el diseño de la página, como la colocación de los elementos y la disposición de las secciones de la página.

Etiquetas de texto

Las etiquetas de texto en HTML son elementos que permiten marcar y estructurar el contenido de una página web. Cada etiqueta tiene su propia función y puede incluir atributos para proporcionar información adicional.
Las etiquetas se organizan en una jerarquía de elementos, donde algunos elementos son hijos de otros. Por ejemplo, el elemento body es el contenedor principal para todo el contenido visible en la página, y dentro del body se pueden incluir elementos como h1, h2, h3, etc. entre otros.
Las etiquetas también pueden ser utilizadas para dar estilo al contenido, utilizando CSS para definir cómo se debe presentar visualmente cada elemento en la página.
En resumen, las etiquetas de texto en HTML son una herramienta fundamental para crear páginas web estructuradas y bien organizadas

Aplicando CSS a los textos

Bien, entonces ya conocemos las etiquetas de texto, pero el texto sin formato es un poco aburrido ¿verdad? veamos que propiedades podemos modificar con css:

  • color: Define el color del texto.
  • font-family: Define el tipo de fuente que se va a utilizar.
  • font-size: Define el tamaño de la fuente.
  • font-weight: Define el grosor de la fuente.
  • font-style: Define el estilo de la fuente (por ejemplo, cursiva).
  • line-height: Define la altura de línea del texto.
  • text-align: Define la alineación horizontal del texto.
  • text-decoration: Define la decoración del texto (por ejemplo, subrayado)
  • text-transform: Define la transformación de texto (por ejemplo, mayúsculas o minúsculas).
  • letter-spacing: Define el espaciado entre caracteres.
  • word-spacing: Define el espaciado entre palabras.
  • text-shadow: Define una sombra detrás del texto
  • white-space: Define cómo se maneja el espacio en blanco dentro del texto.

    
        Sintaxis en Css:

            h1{
            font-size: 20px;
            letter-spacing: 10px;
            font-family: italic; 
            text-decoration: none;
            font-weight: 800;
        }
        
    

    
        también se puede hacer individualmente en html, 
        añadiendo un style al elemento que queremos aplicar estilos

        por ejemplo:

        
            <h1 style="color: rgb(161, 109, 109); font-size: 20px;">
                Texto de ejemplo
            </h1>

Contenedor en HTML

En el contexto de desarrollo web, un div es un elemento HTML que se utiliza para definir una sección o un contenedor genérico dentro de una página web. La etiqueta div se utiliza comúnmente como un contenedor para agrupar elementos HTML juntos y aplicar estilos de CSS a esos elementos.

                

                Un contenedor en HTML se ve de esta manera:

                <div class="ejemplo">

                    <h1>texto aleatorio</h1>
                    <p>motomami motomami motomami</p>

                </div>

Contenedor en CSS

Por ejemplo, si quisieras agrupar un conjunto de elementos HTML (como un título, un párrafo y una imagen) juntos en una sección de tu página, podrías colocarlos dentro de un elemento div. Luego, podrías aplicar estilos CSS a ese div para cambiar el color de fondo, el ancho, el margen, la posición y otros atributos visuales.

                
                .ejemplo{

                    padding: 2%;
                    border-radius: 20px 20px 20px 20px;
                    color: #349595; 
                    background-image: url(ejemplo.png);
                    background-size: 100%;
                }

¿Cómo modificar elementos en el navegador?


Paso 1: ¡Clic!

En nuestra página web, seleccionamos cualquier elemento, en este caso el título de esta página. hacemos clic derecho y seleccionamos Inspeccionar o inspeccionar elemento en Chrome.



Paso 2: Inspector ¿quesessssoooo?

Una vez haciendo clic en inspeccionar, se abrira una ventana similar a esta imagen



Paso 3: Visualizar elementos en HTML

¿Como funciona el inspector? Del lado izquierdo tenemos nuestro codigo HTML. A medida que seleccionemos elementos en el codigo, se iran resaltando elemento correspondiente en la página web. En este caso estamos seleccionando el h1 del título actividad.

Tip: también se puede modificar el texto haciendo doble clic sobre el mismo




Paso 4: Propiedades en CSS

En la parte izquierda del inspector veremos todas las caracteristicas de nuestro elemento en css. Teniendo seleccionado nuestro elemento (Puede ser texto, un contenedor, una imagen etc.) iremos a la parte que dice: element.Style y ahí podremos modificar las carácteristicas.

Aclaración
Estos cambios desaparecerán al recargar la página, pero esta técnica sirve para visualizar los cambios en tiempo real y así poder trabajarlas luego en nuestro editor de código.
