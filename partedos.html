Capitulo 2: Preparando el entorno de trabajo

Por: Félix Achucarro


Para acompañar este recorrido, debes ingresar a la siguiente página para descargar nuestro editor de codigo y las herramientas que utilizaremos durante las siguientes clases:
¡Bienvenidos al desarrollo Web!

Instalación Rápida

Paso 1: Descarga el instalador de la página señalada

Paso 2: Acepte los terminos y todas las etapas de instalación

Paso 3: Busca el programa "visual studio code" en el escritorio o en menú inicio

Ya en el programa

Una vez ingresado al programa, nos aparecerá un menú central con varias opciones.
En primera instancia vamos a elegir la opción "Abrir carpeta"

Creando nuestra carpeta de trabajo

En el escritorio o preferentemente en mis documentos, crearemos una carpeta con la siguiente información:
curso - nombre del alumno

Creando Archivos

Una vez creada la carpeta y seleccionada para abrir en Visual studio Code
vamos a entender como es la interfaz del programa: Por el lado izquierdo tenemos una barra con iconos que nos índica en donde estamos trabajando. En este caso siempre estaremos en el explorador de archivos
El explorador de archivos, como índica su nombre es un visualizador de la carpeta en la cual trabajaremos. podremos crear distintos archivos de distintos lenguajes de programación y también arrastrar elementos multimedia.

en primer instancia: crearemos un archivo nuevo y lo nombraremos "index.html" y también uno llamado "styles.css"

Estructura Html

Vscode tiene atajos del teclado muy interesantes, lo que nos permite ahorrar mucho tiempo escribiendo codigo, para generar una estructura html completa, solo hay que poner un signo de exclamación (!) y y apretar enter en la primer opción

Se genera automaticamente una estructura html con sus etiquetas head y body

Live server


Paso 1:

Vamos a instalar una extensión de vscode llamada "live server", es una extensión que permite visualizar en el navegador y en tiempo real los cambios que realicemos en nuestra página web


En la barra de iconos de la izquierda vamos a buscar el icono de extensiones



Paso 2: Búsqueda

En la barra de búsqueda de la sección buscamos Live server y nos debería aparecer el complemento



Paso 3: Instalación

En el menú del complemento, buscamos el botón con la opción instalar y listo, ya tenemos live server instalado.




Paso 4: Prueba de estado

Para comprobar que Live server este funcionando, vamos a escribir una etiqueta h1 con un texto. Luego vamos a guardar los cambios con control(ctrl) + S
Para ver que la aplicación funciona, vamos a hacer clic derecho con el mouse en alguna parte del codigo y seleccionar la opción Open with live server. Se nos abrirá una pestaña del navegador con nuestra página web.
